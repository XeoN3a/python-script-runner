# Finished Package

This is the completed python-script-runner package.

## Contents
- Python executable (.exe)
- avalible libraries:
-   🧠 Core / System
-   sys – Python runtime control (args, path, exit)
-   os – files, folders, system commands
-   platform – system info (Windows/Linux/etc)
-   subprocess – run external programs
-   shutil – file operations (copy/move/delete)
-   pathlib – modern file path handling

-   ⏱ Time / Scheduling
-   time – sleep, timestamps
-   datetime – dates and times
-   calendar – calendar utilities
-   threading – run tasks in parallel
-   asyncio – async programming

-   📁 Files / Data
-   json – JSON read/write
-   csv – CSV files
-   pickle – save Python objects
-   configparser – config files (.ini style)

-   🔢 Math / Science
-   math – math functions
-   random – random numbers
-   statistics – stats tools
-   decimal – precise decimals
-   fractions – fractions support

-   🌐 Internet / Networking
-   socket – low-level networking
-   urllib – web requests
-   http – HTTP tools
-   ftplib – FTP access
-   ssl – secure connections

-   🧰 Utilities
-   re – regex (text patterns)
-   uuid – unique IDs
-   hashlib – hashing (MD5, SHA)
-   base64 – encoding/decoding
-   copy – object copying

-   🖥️ GUI / UI
-   tkinter – basic GUI system

-   🧪 Debug / Dev tools
-   logging – logging system
-   traceback – error details
-   inspect – introspection (look into code)
-   warnings – warnings control

-   🧵 System-level helpers
-   gc – garbage collector
-   argparse – command line arguments
-   typing – type hints
-   enum – enums

## Installation
Extract the RAR file and follow the readme.txt files in the folder after extracting.
